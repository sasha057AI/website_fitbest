# FitBest — сайт интернет-магазина

Одностраничный лендинг магазина мебели для салонов красоты (кресла и стулья для мастеров
маникюра и салонов). Статический сайт: HTML + CSS + ванильный JavaScript, без сборщиков.

## Как открыть

- **Локально:** просто откройте `index.html` двойным кликом в браузере.
  - Каталог и модальное окно заказа работают через JavaScript. Если открывать по `file://`,
    в некоторых браузерах скрипт может блокироваться — тогда запустите локальный сервер
    (любой статический хостинг) или загрузите сайт на хостинг.
- **Публикация:** загрузите все файлы на любой хостинг или GitHub Pages — сайт статический.

## Структура

```
index.html            — вся страница (шапка, обложка, преимущества, каталог, «О нас», футер, модалка)
css/styles.css        — все стили; палитра задана переменными в начале файла (:root)
js/main.js            — данные товаров + отрисовка каталога, мобильное меню, модалка заказа
assets/
  logo-fitbest.png    — логотип
  hero.svg            — фон обложки (замените на своё фото hero.jpg + поправьте путь в CSS)
  about.svg           — картинка в блоке «О нас» (замените на своё фото)
  products/p1..p6.svg — картинки товаров (замените на реальные фото)
```

## Как заменить товары (главное)

Все товары каталога — в массиве `products` в начале файла [`js/main.js`](js/main.js).
Каждый товар — один объект:

```js
{ name: 'Rosa', model: 'Кресло мастера маникюра', price: 8900, oldPrice: null, img: 'assets/products/p1.svg' }
```

- `name` — название модели (крупно на карточке)
- `model` — тип/описание (мелким шрифтом)
- `price` — цена в рублях (число, без пробелов)
- `oldPrice` — старая цена для скидки. Поставьте число — появится бейдж «sale» и зачёркнутая
  цена. Поставьте `null` — скидки нет.
- `img` — путь к фото товара

**Добавить товар** — скопируйте одну строку-объект и допишите в массив.
**Удалить товар** — удалите его строку.
**Заменить фото** — положите файл (например `p1.jpg`) в `assets/products/` и укажите его в `img`.

## Как заменить фото обложки и «О нас»

- Обложка: положите фото в `assets/` и в [`css/styles.css`](css/styles.css) в правиле `.hero`
  замените `url('../assets/hero.svg')` на своё (например `../assets/hero.jpg`).
- Блок «О нас»: в `index.html` замените `src="assets/about.svg"` на своё фото.

## Как изменить цвета

В [`css/styles.css`](css/styles.css) вверху блок `:root` — все цвета палитры. Например
`--accent` (пудрово-розовый акцент кнопок и заголовков), `--bg` (фон), `--dark` (футер).

## Контакты и тексты

Телефон, адрес, почта, соцсети и текст «О нас» правятся прямо в `index.html`
(разделы `<header>`, `<section class="about">`, `<footer>`).

## Форма заказа — важно

Сейчас форма заявки в модальном окне **не отправляет данные на сервер** (у статического сайта
нет бэкенда) — она просто показывает «Спасибо». Чтобы получать реальные заявки, подключите
внешний сервис. Самый простой способ — [Formspree](https://formspree.io):

1. Зарегистрируйтесь, создайте форму, получите её endpoint (вида `https://formspree.io/f/xxxx`).
2. В [`js/main.js`](js/main.js) в обработчике `form.addEventListener('submit', ...)` вместо
   заглушки отправьте данные `fetch`-запросом на этот endpoint.

Аналогично можно подключить Google Forms, Тильда-формы или собственный бэкенд.
Готов помочь с этим отдельным шагом.
